## Deskripsi

Sesuai instruksi Tugas Mandiri, langkah pertama yang ku lakuin adalah hapus elemen tombol dan referensi fungsi "Paragrafkan" dari  index.html dan index.js biar antarmuka hanya menyisakan dua tombol utama.

Selanjutnya, tambahkan logika untuk menghitung secara dinamis jumlah huruf kecil dan huruf besar. Pada index.js, sy membuat sebuah fungsi bernama updateHitungan() yang akan membaca tiap karakter di dalam kotak input menggunakan loop.

Berikut potongan logikanya:

```
function updateHitungan() {
    let teks = editorElement.value;
    charCountElement.textContent = teks.length;
    
    let jmlKecil = 0;
    let jmlBesar = 0;
    
    for(let i=0; i < teks.length; i++){
        if(teks[i] >= 'a' && teks[i] <= 'z'){
            jmlKecil++;
        } else if(teks[i] >= 'A' && teks[i] <= 'Z'){
            jmlBesar++;
        }
    }
    
    hkElement.textContent = jmlKecil;
    hbElement.textContent = jmlBesar;
}

```

Jika karakter yang sedang dipindai berada di antara rentang 'a' sampai 'z', maka variabel jmlKecil akan bertambah. Logika yang sama diterapkan buat mengidentifikasi huruf kapital pada jmlBesar. Hasil perhitungan dari kedua variabel tersebut kemudian ditampilin ke dalam elemen span dengan ID #hk dan #hb.

Fungsi perhitungan ini dipanggil di dalam event listener "input" agar angka penghitung berubah secara real-time ketika ada interaksi:

```
editorElement.addEventListener("input", updateHitungan);

```

Terakhir, fungsi updateHitungan() juga ditambahin ke blok fungsi tombol "Besarkan" dan "Kecilkan". Tujuannya biar ketika seluruh teks dikonversi paksa ukurannya melalui tombol, angka jumlah huruf kecil dan besar di layar akan langsung menyesuaikan diri dengan kondisi yang baru.
